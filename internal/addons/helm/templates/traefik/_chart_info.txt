Traefik Helm Chart (Minimal)
============================

Source: https://github.com/traefik/traefik-helm-chart
Repository: https://traefik.github.io/charts

Chart Version: 39.0.0
App Version: v3.6.7
Downloaded: 2026-01-25

This is a minimal version of the Traefik Helm chart containing only
the files needed for standard Kubernetes Ingress operation:
- CRDs removed (using standard k8s Ingress, not IngressRoute)
- Documentation files removed
- Hub templates removed (commercial feature)
- Gateway API templates removed (separate addon available)
- Monitoring templates removed (ServiceMonitor, PrometheusRules)
- Optional features removed (HPA, PVC, TLS options/stores)

Configured to use NodePort on 30000/30001 for Hetzner LB compatibility.
